# Image Provenance Policy

Images in this folder must be traceable.

## WILD Hardware Images

Do not use AI-generated imagery for any WILD device depiction, including:

- Device photographs.
- Device illustrations.
- PCB renders.
- CAD renders.
- Connector diagrams.
- Pinout diagrams.
- Schematics of the actual device.

Use only real photographs, CAD/EDA exports, microscope images, measured drawings, screenshots from project tools, or hand-authored diagrams derived from project source files.

## Conceptual Diagrams

Conceptual workflow diagrams may be hand-authored in Markdown, Mermaid, SVG, or another editable source format. If a visual is not directly traceable to hardware, label it clearly as conceptual and do not make it resemble a real WILD hardware photograph or render.
