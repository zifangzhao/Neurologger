# Examples

Examples should bridge the gap between reference documentation and reproducible experiments.

## Planned Example Set

| Example | Scope |
| --- | --- |
| Basic neural recording | Battery, SD format, BLE connect, recording, export, preprocessing. |
| Closed-loop ripple detection | Filter setup, thresholding, stimulation configuration, validation waveform. |
| Theta phase stimulation | Hilbert mode setup, phase logic, stimulation timing, event export. |
| Multi-animal synchronization | Multiple WILD_console instances, start timing, sync line, analysis alignment. |
| Outdoor recording | Power planning, SD choice, wireless checks, data integrity checks. |
| Camera plus IMU | Camera decode, IMU processing, and behavioral alignment. |

## Example Template

Each example should include:

- Goal.
- Hardware revision.
- Firmware binary.
- Software version.
- Required accessories.
- Step-by-step protocol.
- Expected outputs.
- Troubleshooting notes.
- Example data or a small validation file.
